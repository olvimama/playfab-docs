# PFEntityCloseHandle  

Closes a PFEntityHandle.  

## Syntax  
  
```cpp
void PFEntityCloseHandle(  
    PFEntityHandle entityHandle  
)  
```  
  
### Parameters  
  
**`entityHandle`** &nbsp; PFEntityHandle  
  
Entity handle to close.  
  
  
### Return value
Type: void
  
Result code for this API operation.
