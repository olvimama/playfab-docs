# PFHCIsInitialized  

Returns true if library has been initialized by PFHCInitialize  

## Syntax  
  
```cpp
bool PFHCIsInitialized(  
)  
```  
  
### Parameters  
  
  
### Return value
Type: bool
  
Returns true if this library has been initialized, otherwise returns false.
  
  
## Requirements  
  
**Header:** PFHttpClient.h
